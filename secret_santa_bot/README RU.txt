Пособие по использованию бота для игры Secret Santa 

Задание сдлеано в виде бота в телеграм, поэтому без его включения, работать ничего не будет 
Если хотите использовать свего бота, достаточно поменять ключ API для корректной работы 

КАК ИГРАТЬ ?

- Необходимо запустить бота или попросить это сделать меня, написав в телеграмме, мой ник @vladoosik1
- Необходимо написать боту (@S3cr3tSanta_bot) Check in для регистрации и check out для выхода 
- В игре может учавствовать от 3 до 500 человек. Если все ваши друзья зарегистрировались, напишите start game, после чего бот выдаст вам ник человека, которму вы дарите подарок  
- Напишите stop game для завершения игры 

ВАЖНО ЗНАТЬ 
- Бот использует ваш ник в телеграме в качестве идентификатора. Если у вас нет ника, бот будет выдавать значение undefined, и если будет два или более друга без ников, 
вы просто запутаетесь кто есть кто. Для корректной работы бота всё же необходимо что бы все участники игры были с никами 
- Если вы написали check in два или более раза, ничего страшного не произойдет, бот будет работать корректно
